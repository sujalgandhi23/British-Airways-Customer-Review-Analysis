# British Airways Customer Review Analysis

## Project Overview

This project analyzes British Airways customer reviews to evaluate passenger satisfaction, service quality, and travel experience. Python was used for data preprocessing, while Power BI was used to build interactive dashboards and generate business insights.

## Business Objective

- Analyze customer reviews.
- Measure customer satisfaction.
- Compare traveller experiences.
- Identify service improvement opportunities.
- Build interactive dashboards.

## Tools Used

- Python
- Pandas
- Jupyter Notebook
- Power BI
- DAX
- Power Query

## Dataset

- British Airways Reviews
- Countries Dataset

## Files Included
